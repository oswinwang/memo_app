# memorize

專題的前端

main function在lib資料夾中

* pages中為所有的頁面
  * homepage為主頁面的框架，包含appbar和buttomnavigationbar
  * learningpage1為學習的主頁面可以導頁面至choosepage、importpage1、managepage

* component為一些按鈕的樣式
* model是資料的格式
* service目前存放連接api的功能
* style存放一些按鍵的style
